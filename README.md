## Requirements

- Node.js v8

## Usage

```sh
# install dependencies
npm install

# run app in development on port 8080
npm run dev

# lint
npm run lint

# run all tests
npm test

# run unit tests
npm run test:unit

# run end-to-end tests
npm run test:e2e

# build for production
npm run build
```
